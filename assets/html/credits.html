MapPoster

Une application proposée par Messe Basse Production. Pas de cookies, pas de trackers rien de tout cela, vous naviguez ici en toute sécurité.

Les fonds de cartes sont mis à disposition par OpenStreetMap, GeoPortail, ESRI, ESRI et Mapiful. Notre reconnaissance à l'égard de ces contributeurs est immense et sincère.

MapPoster est une application sous licence libre GPL-3.0 qui utilise les librairies Leaflet, SmoothWheelZoom, html2canvas, jsPDF et KolorPick.

Le code source de l'application est disponible sur le dépôt GitHub suivant, toute contribution, idée de fonctionnalité ou report de bug est le bienvenu !
